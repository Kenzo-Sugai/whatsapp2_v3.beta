Informa Horários Disponíveis

UMLUseCase
MatchMaker :: Model :: Proprietário da Quadra :: Informa Horários Disponíveis

Description

Atores: Proprietário da Quadra

Sumário: Este caso de uso descreve o processo do proprietário que irá disponibilizar os horários de funcionamento de sua Quadra. É necessário que ele especifique quantos horários disponíveis terá por dia e qual será o intervalo de tempo entre eles.

Pré-Condições: Para que o proprietário possa informar os horários disponíveis, é necessário que haja o cadastro da Quadra anteriormente.

Pós Condições: Os horários são salvos no banco de dados, possibilitando que o Cliente verifique o horário ideal para a utilização da Quadra a partir dos horários disponíveis.

Ponto de Inclusão: Não há nenhum Ponto de Inclusão

Ponto de Extensão: Não há nenhum Ponto de Extensão

Fluxo Principal: 1. O Proprietário entra na parte de gestão de sua Quadra. 2. O Proprietário, a partir de uma tabela pré-disponibilizada, seleciona os horários e intervalos que ele quer deixar disponível para cada dia da semana. 3. Após ele selecionar, o sistema ele salva as informações.
