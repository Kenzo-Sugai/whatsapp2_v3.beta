Description

Realizar Agendamento

Atores: Cliente Sumário: Este caso de uso descreve o processo de um cliente realizando uma reserva de quadra através do sistema.

Pré-condições: Para que o cliente possa reservar uma quadra através do sistema, é necessário que ele esteja registrado e tenha feito o login. Além disso, o sistema precisa ter informações atualizadas sobre a disponibilidade das quadras.

Pós-condições: A reserva da quadra é registrada no sistema e a disponibilidade da quadra é atualizada. Além disso, tanto o cliente quanto o dono da quadra são notificados da reserva. Ponto de Inclusão: Não há nenhum Ponto de Inclusão Ponto de Extensão: Não há nenhum Ponto de Extensão

Fluxo Principal: 1.O cliente informa sua localização. 2.O cliente seleciona o tipo de esporte que deseja praticar. 3.O cliente indica se deseja compartilhar a quadra com outros jogadores ou reservar a quadra exclusivamente para ele. 4.O sistema filtra as quadras disponíveis de acordo com as preferências do cliente. 5.O sistema apresenta ao cliente uma lista de dias e horários disponíveis para as quadras selecionadas. 6.O cliente seleciona o dia e horário que deseja reservar. 7.O Cliente realiza o pagamento. 8.O sistema confirma o pagamento e registra a reserva da quadra para o cliente. 9.O sistema envia uma confirmação da reserva para o cliente e para o proprietário da quadra.
