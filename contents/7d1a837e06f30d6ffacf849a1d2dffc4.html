Description

Realizar Cadastro

Atores: Cliente

Sumário: O cliente pode se cadastrar no software para ter acesso aos recursos do software

Pré-condições:
O cliente deve ter um email valido ou número de telefone, também é recomendado ter documentos como RG.

Pós-condições:
As informações são enviadas no banco de dados, possibilitando que o cliente verifique suas informações caso queira editar.

Ponto de Inclusão:
Não há nenhum Ponto de Inclusão

Ponto de Extensão:
Não há nenhum Ponto de Extensão

Fluxo Principal: 1. O Cliente entra na parte cadastro. 2. O Cliente, a partir que ele preencha todas as informações ele pode seguir com o cadastro. 3. Após ele preencher, o sistema ele salva as informações.
